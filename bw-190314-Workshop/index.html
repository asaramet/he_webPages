Open Source / Commercial

Commercial software licensing:

  • Basic restirctions (copyright, right to modify, right to distribute, etc.)
  • Volume licensing (software used on large number of computers)
  • Contractual license program (certain number of licenses over a fixed period of time)
  • Licensing per concurent / floating user (specific number of users have access at the same time)
  • Licensing per server / site (nodes or CPU-hours, regardles of number of users)

University managed license servers

  • Server Hostname / IP
  • Open ports
  • License key / file

Modules package

  • Uni specific software instalation path: /opt/bwhpc/[UNI-PREFIX]/[TYPE]
  • Uni specific modulefiles path: /opt/bwhpc/[UNI-PREFIX]/modulefiles/[TYPE]

Modulefile - file containing Tcl code for the Modules package

Wrapper for passwordless remote shell

$ module load system/ssh_wrapper/0.1

$ export FLUENT_SSH=/opt/bwhpc/common/system/ssh_wrapper/0.1/ssh

SSH tunnel (port forwarding)

  • SSH connection from bwUniCluster to an intermediate server / PC (Port 22)
    • Key based secure authentication
      • $ ssh-keygen
      • $ ssh-copy-id -i ~/.ssh/id_rsa.pub <USER_ID>@<COM_HOSTNAME>
  • License server access from the intermediate PC through dedicated ports

SSH port forwarding command options:

  • -M: Places the ssh client into “master” mode for connection sharing
  • -S ctl_path: Location of a control socket for connection sharing
  • -L [bind_address:]port:host:hostport: Port and host on the server to be redirected to local hostport
  • -N: Do not execute a remote command
  • -n: Prevents reading from stdin
  • -f: Requests ssh to go to background just before command execution
  • -T: Disable pseudo-terminal allocation
  • -O ctl_cmd:Control an active connection multiplexing master process
    • check: check that the master process is running
    • exit: request the master to exit
