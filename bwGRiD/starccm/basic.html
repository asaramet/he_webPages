STAR-CCM+ on bwGRiD Cluster, Esslingen

Accessing and basic usage

To view which STAR-CCM+ modules are available to use on the cluster, run the following command: 

$ module avail esslingen/star-ccm+

To load the appropriate version (if available), run: 

$ module load esslingen/star-ccm+-<version>
where <version> is the desired available STAR-CCM+ version to load.
