Location Own Licenses

(Standort-eigene Lizenzen)

Private. Owned by each university.

Available only for the users of a specific university.

Soft installed in the university's folder on bwUniCluster.

/opt/bwhpc/es

Prior clarification by software manufacturers required, whether site-related licenses are allowed on bwHPC clusters.

May need extra IP connections / ports open on bwUniCluster.

Firewall rights should be modified on site, to allow connection to bwHPC clusters.

Limited availability due to high latency. Waiting time on the license server should be increased.

IP-Based licenses

Uses TCP/IP sockets to communicate.

Sets a default license manager port.

Modify port on the server or IP-forwarding.

On client set ip/hostname and port.

Firewall configured to allow communication on the defined port.

MAC Address Based licenses

Licenses tied to particular MAC addresses.

A Key file is provided for clients to register.

Changing hardware/MAC address on client site, leads to modification on license server.

Requires: IP, Port, Key-file.

Power On Demand

Run hours offered at user demand.

Available to use on HPC clusters.

It counts only the walltime of a running job.

User specific POD-key.

export CDLMD_LICENSE_FILE=1999@flex.cd-adapco.com

export LM_PROJECT=vuy...[POD-key]

License Manager

University managed licenses.

Usually floating licenses.

Limited number of licenses, shared among large number of users.

University defines the pool of supported licenses.

Software comes with an install manager for license server.

Access to server's IP address, specific ports.

Site firewall allows access to bwUniCluster.

FlexNet (FlexLM)

License server host. E.g.: lmserver.uni.de

Default ports: 1055, 2325

License client file: $base_dir/shared_files/licensing/ansyslmd.ini

SERVER=1055@lmserver.uni.de

ANSYSLI_SERVERS=2325@lmserver.uni.de

LS-DYNA Network License

License server host or ip address. E.g.: ls-dyna.uni.de

Default port: 31010

License client file: $base_dir(or $local_dir)/LSTC_FILE

#LICENSE_TYPE: network

#LICENSE_SERVER: ls-dyna.uni.de

Environment variables. (Priority)

setenv LSTC_LICENSE: network

setenv LSTC_LICENSE_SERVER: ls-dyna.uni.de

(bw-Landeslizenzen)

Central service managed by KIT.

Available for all the users from BW universities.

Soft installed in the global folder on bwUniCluster.

May be limited to specific university's license quota on bwUniCluster.

License Manager

KIT managed licenses.

License server at KIT.

Floating licenses (FlexNet).

A certain number of licenses available for all bwHPC users.

May have to wait in the license queue.

Job may be canceled if no free licenses are available at the moment.

License Quota Manager

Licenses partially available.

Every bwHPC site manages its license server.

Users on bwUniCluster may have to buy a license quota beforehand.

Limited availability.

E.g.: MATLAB - only single node jobs up to 16 cores maximum.
